This folder is designated for custom prayer call (Azan) and Iqamah sound assets.

To play real voice Azan and Iqamah sounds, place the following audio files in this directory:
1. `azan-makkah.mp3` - Plays when "Makkah style Azan" (Azan 1) is selected.
2. `azan-madinah.mp3` - Plays when "Madinah style Azan" (Azan 2) is selected.

If these files are absent, the application automatically falls back to pleasant synthesized oscillator notes, ensuring you always hear alerts and no errors occur!
